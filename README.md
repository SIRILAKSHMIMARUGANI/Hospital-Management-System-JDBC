# Hospital-Management-System-JDBC
Hospital Management System using Java, JDBC, and MySQL


The Hospital Management System is designed to reduce manual work and improve the management of hospital data. Using JDBC, the application connects Java with the MySQL database, allowing smooth storage, retrieval, updating, and deletion of records.

🔹 Key Features

Patient Registration and Management
Doctor Information Management
Appointment Booking System
View Patient and Doctor Details
Update and Delete Records
Secure Database Connectivity using JDBC
Efficient Data Storage with MySQL

🔹 Technologies Used

Java
JDBC
MySQL
SQL Queries
Object-Oriented Programming Concepts
